## Project Overview

This project presents a simulated NIST Cybersecurity Framework (CSF) Gap Analysis conducted for a mid-sized retail financial services organization (~1,200 employees) undergoing its first formal compliance assessment.

## Files in This Repository

| File | Description |
|---|---|
| `gap_analysis.md` | NIST CSF gap analysis across all 5 functions |
| `risk_register.md` | Risk register with likelihood, impact, and mitigation |
| `requirements.md` | Compliance requirements traceability matrix |
| `incident_response_process.md` | As-Is / To-Be incident response process map |
| `assets/compliance_dashboard.png` | Power BI compliance posture dashboard |
